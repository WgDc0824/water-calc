# 压力电耗快速计算工具 (Water-Calc)

这是一个基于出厂压力自动计算各项能效指标的网页小工具。

## 🚀 在线使用
点击下方链接即可直接在浏览器（手机/电脑）中使用：
👉 **[点击这里开始使用工具](https://WgDc0824.github.io/water-calc/)**

## 📊 计算逻辑说明
本工具根据压力值 ($P$, 单位: MPa) 自动推算以下四项关键指标：

1. **理论最小千吨水电耗 (kWh/kt)**: $P \times 277.95$
2. **系统效率 50%**: 理论值 / 0.5
3. **系统效率 60%**: 理论值 / 0.6
4. **系统效率 70%**: 理论值 / 0.7

## 📱 使用小技巧
* **手机快捷访问**：在手机浏览器中打开链接，选择“添加到主屏幕”，即可像 App 一样随时点击使用。
* **数据精度**：目前计算结果默认保留 4 位小数。
